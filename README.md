### 기술 사용 이유

* CRA : react 를 가장 간편하고 빠르게 세팅하기위해 Create-react-app 보일러 플레이트를 사용했습니다.

* react-router : todo 의 상세페이지를 랜더링하고, 누른 순서에 따라 뒤로가기시 조회를 위해서 router를 사용했습니다. 그리고 토큰 유무에 따라 페이지를 redirect 하기 위해서 사용했습니다.

* axios : 제가 사용하기 가장 익숙한 api 라이브러리로서 사용하였습니다. 또한 통신시 header에 토큰을 넣기위해 사용했습니다.
